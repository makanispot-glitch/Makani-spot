📖 دليل الربط بالموقع
1
افتح Apps Script
من الشيت: Extensions → Apps Script
تأكد وجود doGet() و doPost() و approveSpace()
2
انشر كـ Web App
Deploy → New Deployment → Web App
• Execute as: Me • Who has access: Anyone
⚠️ كل تعديل في الكود → New Deployment مش Update
3
ضع الرابط في الداشبورد
انسخ الرابط وضعه في خانة "رابط Apps Script" في الـ Sidebar ← اضغط "تحميل البيانات"
4
اربط الموقع الرئيسي
في app.js — أول الملف:
const SHEET_URL = "https://script.google.com/macros/s/AKfy.../exec";
✅ بعد الحفظ — كل مساحة تمت الموافقة عليها تظهر فوراً على الموقع
5
نظام الموافقة
1. أضف مساحة من الداشبورد → تُحفظ بـ ظاهر = لا
2. اذهب لصفحة موافقة المساحات → اضغط ✅ موافقة
3. المساحة تظهر فوراً على الموقع
6
رفع الصور الصحيح
استخدم postimages.org للرابط المباشر
